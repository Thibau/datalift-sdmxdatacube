## Démonstration de SDMXDataCube

Le dossier sdmxdatacube/demo contient deux jeux de données pouvant être utilisés pour tester ce module. Ces jeux de données doivent être importés dans Datalift pour que le module devienne accessible (ne pas oublier de forcer le rafraîchissement de la page pour faire apparaître le bouton d'exécution du module).
